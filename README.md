# karnataka-telangana-mandalika-padalu
కర్ణాటక తెలంగాణ సరిహద్దు ప్రాంతాలైన నారాయణపేట పరిసర గ్రామాల్లో అందమైన మాండలిక పదాలు కనిపిస్తాయి. వాటిని రాబోయే తరాలకు అందించ డానికి ఆ ప్రాంత పదాలతో ఒక డిజిటల్ నిఘంటువు రూపొందిస్తున్నాం 
# కర్ణాటక తెలంగాణ సరిహద్దు ప్రాంత నారాయణపేట మాండలిక నిఘంటువు (Narayanapet Dialect Dictionary)

తెలంగాణలోని నారాయణపేట జిల్లా సరిహద్దు గ్రామాల్లో వాడే అరుదైన మాండలిక పదాలను భద్రపరిచే ఉద్దేశంతో ఈ ప్రాజెక్ట్ ప్రారంభించబడింది.

## ప్రాజెక్ట్ లక్ష్యం:
- ప్రాంతీయ మాండలిక భాషను డిజిటలైజ్ చేయడం.
- AI (Artificial Intelligence) పరిశోధనలకు తెలుగు మాండలిక డేటాను అందించడం.
- భాషా ప్రేమికులకు మరియు విద్యార్థులకు మాండలిక పదాల అర్థాలను సులభంగా అందుబాటులో ఉంచడం.

## ఉదాహరణ పదాలు:
| పదం | అర్థం | సందర్భం |
| :--- | :--- | :--- |
| **గూటం** | గుంజ | వాడు గూటం లాగా కదలకుండా నిల్చున్నాడు. |
| **కూనం** | గుర్తు | బిడ్డా నిన్ను జూసి చాన్నాళ్ళయిందిగా అందుకే కూనం పట్టలే. |
| **రెట్ట** | భుజం | వాడు నన్ను రెట్టవట్టి లాగిండు. |
